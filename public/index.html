FreeCodeCamp BackEnd API project

Image search API

    User stories

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Example of Usage

passing this url:

https://andrei-little-url.herokuapp.com/https://www.google.com

you will get this:

https://andrei-little-url.herokuapp.com/1

Example output

{ {'url': 'https://www.google.com', 'urlShortned': 'https://andrei-little-url.herokuapp.com/1' } }
