FreeCodeCamp BackEnd API project

Image search API

    User stories

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Example of Usage

Searching the image:

You can search something and get a list of items from the search with the Image URL, page URL and alt text:

http://andrei-image-search.herokuapp.com/search/dogs

Skipping pages

You can skip pages by passing ?offset={number of pages}, ex:

http://andrei-image-search.herokuapp.com/search/dogs?offset=2

Search history

You can also get the history of the searches ordered by the most recent:

http://andrei-image-search.herokuapp.com/history
